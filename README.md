# Enterprise Fine-tuning & Custom Models

> **Track:** Anthropic Enterprise Implementation (Track 1)
> **Course 4 of 6**

Prepare datasets, run fine-tuning jobs, and evaluate custom Claude models for enterprise-specific tasks.

---

## Modules

### Module 1: When to Fine-tune vs. Prompt

**Learning Objectives:**
- Identify use cases requiring fine-tuning
- Calculate cost-benefit of fine-tuning
- Set baseline performance benchmarks

### Module 2: Dataset Preparation & Curation

**Learning Objectives:**
- Structure training examples in JSONL format
- Clean and deduplicate datasets
- Balance positive and negative examples

### Module 3: Fine-tuning Pipeline Setup

**Learning Objectives:**
- Submit fine-tuning jobs via API
- Monitor training progress
- Handle dataset validation errors

### Module 4: Evaluation & Deployment

**Learning Objectives:**
- Run holdout set evaluation
- A/B test fine-tuned vs. base model
- Deploy fine-tuned model to production
